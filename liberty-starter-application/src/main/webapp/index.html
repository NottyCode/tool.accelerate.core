mvn install liberty:run-server
Access your application at:

Your Liberty server can be found in:
target/wlp/usr/servers/LibertyProjectServer

Once the download is complete, locate your project and unzip it.

Your application code can be found in:
src/main/java
On the LibertyProject pom.xml run:
mvn install -Dcf.org=[your email address] -Dcf.username=[your username] -Dcf.password=[your password]
Access your application at:
LibertyProject.eu-gb.mybluemix.net/myLibertyApp/
To manage your deployed application visit your dashboard on Bluemix.
Optional, import downloaded Maven into the WebSphere Developer Tool
  • Step 1

    Go to the WebSphere Development Tool and Import "Existing Maven Projects".

  • Step 2

    Select the Maven projects you want to import.

  • Step 3

    Once the project is imported, make changes to the custom code and build your app.

  • Step 4

    Pickup code and run either: Locally or using Bluemix.

  • Step 5

    Lastly, navigate to this URL on your browser to access your app - http://localhost:9080/myLibertyApp/.
