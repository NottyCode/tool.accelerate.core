Getting started...

  1. Select the technology types you want
  2. Click Download project and unzip the project
  3. Your application code can be found in myProject-application/src/main/java
  4. Run $ mvn install on the LibertyProject pom.xml
  5. Access your application at http://localhost:9080/myArtifactId-application-1.0-SNAPSHOT
  6. Your Liberty server can be found in myProject-wlpcfg/target/wlp/usr/servers/LibertyProjectServer
